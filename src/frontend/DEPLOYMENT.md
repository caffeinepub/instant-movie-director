# Deployment Guide - Instant Movie Director

This guide provides step-by-step instructions for deploying **Instant Movie Director** to the Internet Computer, both locally for development and to the public mainnet.

## Prerequisites

Before deploying, ensure you have the following installed:

- **dfx** (Internet Computer SDK) - Install from [https://internetcomputer.org/docs/current/developer-docs/setup/install](https://internetcomputer.org/docs/current/developer-docs/setup/install)
- **Node.js** (v18 or later)
- **pnpm** package manager
- A code editor and terminal access
